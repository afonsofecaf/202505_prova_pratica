🧪 Prova Prática – QuickSort em Rust

📌 Situação:

Você deve implementar o algoritmo de ordenação QuickSort em Rust para o seguinte array predefinido: 

let mut arr = [34, 7, 23, 32, 5, 62, 31, 12, 43, 3];

✅ Tarefas:

  1. Nível 1: Utilize a biblioteca padrão do Rust para ordenar o array.
    Dica: Utilize métodos como sort() ou outras funções utilitárias do módulo std::collections.
  2. Nível 2: Implemente o algoritmo QuickSort do zero, criando uma struct e implementando as funções de ordenação dentro de um bloco impl.

📝 Requisitos obrigatórios:

  • No Nível 1, o código deve ser direto, utilizando funções nativas.
  • No Nível 2, é obrigatório o uso de struct e impl para organizar o algoritmo QuickSort.
  • O programa deve imprimir o array original e o array ordenado após cada execução.
  • Comente o código explicando o funcionamento das partes principais.

🚀 Exemplo de saída esperada:

Array original: [34, 7, 23, 32, 5, 62, 31, 12, 43, 3]
Array ordenado: [3, 5, 7, 12, 23, 31, 32, 34, 43, 62]

📦 Entrega:

Entregue um arquivo chamado main.rs contendo as duas implementações (Nível 1 e Nível 2), bem organizadas e comentadas.
